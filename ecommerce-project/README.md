# 📊 E-Commerce Data Analysis Project

![Dashboard](powerbi/dashboard.png)

## 🚀 Project Overview
This project focuses on analyzing e-commerce transaction data to extract meaningful business insights.  
The workflow includes data cleaning, SQL analysis, and dashboard visualization.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy) → Data Cleaning
- PostgreSQL → Data Analysis
- Power BI → Dashboard & Visualization

## 🧹 Data Cleaning (Python)
- Handled missing values
- Fixed data types (price, quantity, date)
- Standardized categorical values
- Removed duplicates
- Recalculated total_spent

---

## 📊 SQL Analysis (PostgreSQL)
- Total Revenue & Orders
- Top Customers Analysis
- Category-wise Revenue
- Payment Method Distribution
- Monthly Sales Trend
- Customer Behavior Analysis

---

## 📈 Dashboard Insights (Power BI)
- Total Revenue: **1.57M**
- Avg Order Value: **124.46**
- Top Categories: Butchers, Electronics
- Most Used Payment: Cash & Digital Wallet
- Seasonal Trends observed in sales
